Comment lancer :

1 ere solution :
    Lancer le .jar
2 eme solution :
    Importer le projet dans intellij avec gradle
3 eme solution :
    Compiler les sources avec la librairie JDBC pour MYSQL


Base de données :
    Nous utilisons une base de données stocker sur un serveur distant, elle vous y est accessible depuis internet
    (Vous avez les identifiants de connection dans la class ConnectionSingleton),Si vous n'avez pas internet vous avez
    le fichier creation.sql qui comporte toute la base de données;
